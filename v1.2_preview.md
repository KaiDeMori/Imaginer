# v1.2 Features (so far)

## Major Features

### Drag-and-Drop & Metadata Support
- Implemented drag-and-drop functionality for PNG images.
- Added capability to read metadata from dropped images.
- Fixed timestamp precision for PNG metadata creation.

### Moderation Error Handling
- Created a dedicated moderation error page featuring random error images and animations.
- Added a **"Use Prompt"** button to error placeholders, allowing users to easily retrieve the prompt that triggered an error.

### Gallery Management
- Added a **"Clear Gallery"** button to remove all images at once.
- Implemented a confirmation prompt for the clear action to prevent accidental deletion.
