OpenAI API Documentation Overview
================================

This document provides a compact overview of the OpenAI API documentation with direct links to key topics, focusing on the Responses API and Image generation. It omits advanced features like fine-tuning.

**Core Getting Started**
- **Quickstart:** https://developers.openai.com/api/docs/quickstart : Quick steps to make your first API request and authenticate.
- **Models overview:** https://developers.openai.com/api/docs/models : Guidance on model selection (e.g., GPT-5 family).

**Responses API (primary text & structured responses)**
- **Responses API guide:** https://developers.openai.com/api/docs/guides/responses-api : High-level guide to the Responses API and migration notes.
- **Responses API reference:** https://developers.openai.com/api/reference/resources/responses : Full reference for the `responses` endpoints, request/response schemas and examples.
- **Streaming responses:** https://developers.openai.com/api/docs/guides/streaming-responses : How to receive streaming outputs from the Responses API for lower latency and partial outputs.
- **Webhooks & background mode:** https://developers.openai.com/api/docs/guides/webhooks and https://developers.openai.com/api/docs/guides/background : Options for async handling of Responses API outputs.
- **Structured outputs / JSON schema:** https://developers.openai.com/api/docs/guides/structured-outputs : Techniques for getting reliably structured JSON data from model responses.

**Image Generation & Vision**
- **Image generation guide:** https://developers.openai.com/api/docs/guides/image-generation : Guides to generate images (prompts, parameters, workflows).
- **Images API reference:** https://developers.openai.com/api/reference/resources/images : Endpoint reference for image generation, editing, and variations.
- **Images & vision guide:** https://developers.openai.com/api/docs/guides/images-vision : Using image inputs/outputs, model vision capabilities, and examples (analyzing or generating images).

- Images reference: https://developers.openai.com/api/reference/resources/images
- Images & vision: https://developers.openai.com/api/docs/guides/images-vision

Notes
- This overview emphasizes the Responses API (text/structured outputs, streaming, webhooks) and Image generation (create/edit/vision). Advanced topics (fine-tuning, evals, model optimizing pipelines) are intentionally omitted.
