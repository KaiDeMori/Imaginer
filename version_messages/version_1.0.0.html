🎉 Welcome to Imaginer 1.0! 🎉

Here’s what’s fresh in this release:

  • 🗂️ Download all images in the current gallery as a zip file for backup.
  • 🖌️ Mask mode: brush-based mask painting and erasing.
    This button is hidden by default — enable in advanced settings.
  • Privacy Mode: All metadata is stripped from the images and only your text prompt is added. Enabled by default in advanced settings.

If you’re upgrading from the alpha version, some features may have changed. Please report any issues you find!
