# AI Image Generation Prompts for Early Universe Formation

A collection of prompt ideas for generating transparent PNG assets for the cinematic step.

No need to mention alpha since that is handled via an API parameter.

---

## Nebulae

### Master Prompt
"A vibrant cosmic nebula, glowing, with soft edges, wispy and ethereal"

### Sticker Page (3x3 Grid) Prompt
"A sticker page of 9 unique vibrant cosmic nebulae, each with soft edges and glowing colors, arranged in a 3 by 3 grid on a transparent background. Each nebula should have no white border, no outline, and should fade smoothly into transparency at the edges. No drop shadow, no glow, no border. Each nebula separated with clear space for easy cropping, PNG with alpha channel."

### Sticker Page (3x3 Grid numbered) Prompt
"Using the provided character image as a reference, create a detailed visual character sheet consisting of 9 distinct images arranged in a 3x3 grid. Each image should depict the character clearly and consistently, showcasing different angles, expressions, poses, and details. Include the following:

1. Front full-body view (neutral pose)
2. Back full-body view
3. Side profile view
4. Close-up portrait (neutral expression)
5. Close-up portrait (expressive or emotional)
6. Action pose demonstrating character personality or abilities
7. Detailed view of clothing or armor design
8. Detailed view of accessories, weapons, or special items
9. Character in an environment or context relevant to their story or theme

Maintain consistent art style, color palette, and character design throughout all images. Ensure clarity, high-quality rendering, and visual coherence."
