Box-Shadow Anti-Aliasing Test

This demonstrates how a tiny box-shadow prevents color bleeding on semi-transparent text.

Side-by-Side Comparison

box-shadow: 0 1px 4px rgba(0, 0, 0, 0.04)
No box-shadow

Look closely at the text edges. The button with shadow should have crisp white text, while the one without may show color fringing/bleeding.

Toggle Test

Click the button to toggle the box-shadow on/off and observe the text rendering change.

Technical Details

  • Both buttons have opacity: 0.85
  • Both have white text on gray background
  • Only difference: tiny box-shadow creates new stacking context
  • This changes how browser composites the opacity
  • Result: Different text anti-aliasing behavior
