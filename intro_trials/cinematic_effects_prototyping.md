# Cinematic Effects Prototyping

This file is for experimenting with individual visual effects for the Imaginer welcome cinematic. For each effect, we will note its purpose, a brief description, and reference the example JS/HTML file(s) used for prototyping.

---

## Effect Prototypes


### 1. Starfield Animation
- **Description:** Animated starry background with twinkling and subtle zoom.
- **Example File:** `intro_trials/cinematic_starfield_demo.html`


### 2. Explosion Effect
- **Description:** Randomly placed, animated color explosions (bursts) for the "birth of creativity" scene.
- **Example File:** `intro_trials/cinematic_explosion_demo.html`


### 3. Shake Effect
- **Description:** Screen shake using CSS `@keyframes` for dramatic impact.
- **Example File:** `intro_trials/cinematic_shake_demo.html`


### 4. Gradient Universe Transition
- **Description:** Animated gradient and contrast transitions to simulate universe formation.
- **Example File:** `intro_trials/cinematic_gradient_transition_demo.html`


### 5. Alien Character Animation
- **Description:** Simple animated alien character for the final reveal.
- **Example File:** `intro_trials/cinematic_alien_demo.html`

Add new effects and reference files as needed. Each prototype should use loose_snake_case for all file and variable names.
